# Resume Analyzer Web App

An AI-powered Resume Analyzer web application that parses resumes and provides
structured insights using Python, NLP, and Streamlit.

## Features
- Upload resume (PDF)
- Extract skills & experience
- NLP-based analysis
- Simple web interface

## Tech Stack
- Python 3.10
- Streamlit
- SpaCy
- Pandas

## How to Run
1. Install Python 3.10
2. Install dependencies:
   pip install -r requirements.txt
3. Run app:
   streamlit run app.py
